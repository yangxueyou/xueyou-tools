# 点

```js
<Point
  position="year*value" // 线上的点对应的横纵坐标
/>
```

# 线

```js
<Line
  position="year*value" // 线 对应 的 横坐标、纵坐标
/>
```

# 提示信息

```js
<Tooltip
  showCrosshairs  // 是否显示辅助线，点图、路径图、线图、面积图默认展示
  follow={true}   // 提示信息是否跟随鼠标
/>
```

# 坐标轴（x、y）展示个数

```js
// scale:{} year 对应 x 轴，tickCount 对应 x 轴展示个数（包含数据第一个最后一个，其它的自动计算）
// <Axis /> name 是线对应的内容
// <Geom /> position="year*value"  对应 x、y 轴坐标
// <Geom /> shape="smooth" 线条变成平滑的
const scale = {
  year: {
    type: "linear",
    tickCount: 5,
  },
  // value 是 y 轴，展示范围 0-1，展示个数 10，
  value: {
    type: "linear", // 连续的数据值
    min: 0,
    max: 1,
    tickCount: 10,
  },
};

<Chart
  appendPadding={[10, 0, 0, 10]}
  autoFit
  height={500}
  data={data}
  onLineClick={console.log}
  scale={scale}
>
  <Geom type="line" position="year*value" shape="smooth" />
  <Axis name="value" />
</Chart>;
```

# Chart 图标介绍

```js
<Chart
  appendPadding={[10, 10, 10, 10]} // 再padding的基础上增加的调整边距。
  autoFit // 图表大小自适应，对外层容器的宽和高都会适应。
  height={500} // 高度
  data={data} // 数组对象结构的数据源
  scale={{
    value: { min: 0, alias: "人均年收入", type: "linear-strict" },
    year: { range: [0, 1] },
  }} //  x 轴  y 轴 展示的内容
>
  <Line
    position="year*value" // 线对应的 x y 轴
  />
  <Point
    position="year*value" // 点对应的 x y 轴
  />
  <Tooltip
    follow={true} // 提示信息是否跟随鼠标
  />
</Chart>
```
