## Provider

```js
import { useSelector, useDispatch, Provider } from 'react-redux';

// 不要在 Provider 这个文件夹直接使用 useDispatch，否则会报错，需要包一层
// 因为代码先执行上面，但 useDispatch 需要在被包裹的组件里使用
<Provider store={store}>
  <App {...props}></App>
</Provider>
```
